# BS Prep Tamil Platform

Community preparation platform for the IIT Madras BS Degree Program, maintained by the IITM BS Tamil community. Provides study resources, discussions, and peer guidance in Tamil.

## Setup

```bash
git clone https://github.com/IITMBSTamilCommunity/bsprep
cd bsprep
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in a browser.
